ℹ️ Hướng dẫn

Thanh toán Binance Pay được xác minh bằng cách đọc lịch sử Pay History của chính tài khoản Binance của bạn (không cần đăng ký Binance Pay Merchant).

  1. Vào Binance → Account → API Management → tạo API Key mới.
  2. Chỉ bật quyền "Enable Reading".
  3. Dán API Key/Secret Key vào đây và nhấn "Test connection".

Khách hàng sẽ chuyển đúng số USDT tới Binance ID của bạn và gửi TXID để hệ thống tự đối chiếu.

🟨 USDT BEP20 (BSC)
Đang tải...
Nhận USDT qua mạng BNB Smart Chain (BEP20)
Mặc định: USDT BEP20 chính thức
Lấy tại bscscan.com (miễn phí)
ℹ️ Hướng dẫn BEP20

Hệ thống sẽ tự động theo dõi USDT đến địa chỉ ví của bạn trên BNB Smart Chain.

Ghi chú quan trọng:

  • Mỗi đơn có số USDT duy nhất (offset nhỏ) để phân biệt các lệnh đồng thời
  • BSCScan API Key giúp quét chính xác hơn (khuyến nghị)
  • Nếu không có BSCScan API, worker dùng RPC fallback (giới hạn)
  • Cần ít nhất 12 confirmations BSC ≈ ~36 giây

Mặc định: USDT TRC20 chính thức
Lấy tại trongrid.io (free tier đủ dùng)
ℹ️ Hướng dẫn TRC20

Hệ thống sẽ tự động quét USDT đến địa chỉ ví TRON của bạn qua TronGrid API.

Ghi chú quan trọng:

  • TRC20 phí giao dịch rất thấp (<$1) — phổ biến ở VN
  • Thời gian xác nhận: 20 blocks ≈ ~60 giây
  • TronGrid free tier: 1000 req/day — đủ cho lưu lượng nhỏ
  • Nâng lên TronGrid Pro nếu có nhiều đơn đồng thời
