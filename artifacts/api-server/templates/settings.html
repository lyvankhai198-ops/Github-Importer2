ℹ️ Hướng dẫn BEP20

Hệ thống sẽ tự động theo dõi USDT đến địa chỉ ví của bạn trên BNB Smart Chain.

Ghi chú quan trọng:

  • Mỗi đơn có số USDT duy nhất (offset nhỏ) để phân biệt các lệnh đồng thời
  • BSCScan API Key giúp quét chính xác hơn (khuyến nghị)
  • Nếu không có BSCScan API, worker dùng RPC fallback (giới hạn)
  • Cần ít nhất 12 confirmations BSC ≈ ~36 giây
